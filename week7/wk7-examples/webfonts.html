Using webfonts

This content will be used to demo some webfonts and Font Awesome (icon font).

Reminder about sites to have on your radar:

  • CSS Tricks

    CSS Tricks has a lot of useful articles and guides for various CSS properties.

  • Smashing Magazine

    Smashing Magazine is a design-focused resource. There are many articles about design principles, UI/UX, and CSS.

  • MDN CSS Reference

    This is the Mozilla Developer Network's CSS reference. This is a useful resource for a deeper dive into individual CSS properties as you can find the possible values you can use for each property.
